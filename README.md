# Outil de notes de frais avec ESS'poir

## Contexte

Nous venons de changer de cabinet comptable et sommes avec [ESS'poir](https://www.les-scic.coop/ess-poir)

Il y a un process pour faire les notes de frais qui est un peu laborieux

1. Remplir le fichier excel qui liste les dépenses. Renommer le fichier avec le nom correspond au bénéficiaire et pour période souhaitée. 
2. Rassembler les justificatifs
3. Faire un unique PDF avec le fichier excel et les justificatifs.
4. Déposer le document dans la plateforme partagée

C'est beaucoup de manipulations de fichiers

On peut faire plus simple. C'est l'objet de cet outil


## Installer, lancer en local

Nous recommandons d'utiliser [pnpm](https://pnpm.io/) et sinon, npm fonctionnera très bien

```sh
pnpm install
pnpm run dev
```
